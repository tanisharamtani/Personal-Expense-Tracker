# Person 3 Backend Work

This folder contains the separate Person 3 backend API layer for the Personal Expense Tracker project.

## My Role

Person 3 is responsible for:

- Node.js and Express server setup
- API routes and controller logic
- JWT authentication middleware
- Login, register, forgot password, and reset password APIs
- Expense CRUD APIs
- Budget manager APIs
- Savings goal APIs
- Dashboard summary API
- Analytics and report API
- Error handling and protected routes

## Main Files Added

- `server.js`
- `package.json`
- `controllers/`
- `routes/`
- `middleware/`
- `utils/`

## API Base URL

```txt
http://localhost:5000/api
```

## Run Backend

```bash
npm install
npm run dev
```

Create a `.env` file using `.env.example` before running the backend.
